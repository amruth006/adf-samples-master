Glassfish Extension Documentation

by : Shay Shmeltzer

This extension adds three buttons to JDeveloper's toolbar as well as a main menu option called Glassfish. The buttons will allow you to start, stop and start in debug mode a Glassfish instance.

You can then use the normal one click deploy, and remote debug options to work with the glassfish instance.

In order to get the extension to work, you'll need to configure the start/stope/startDebug command line options in the tools->preferences->Glassfish section.

The buttons in the toolbar will simply invoke those commands.

If your startup script is more complex you can also point the configuration to .bat files that will be run instead of the command lines.

For more information on this extension please see my blog: https://blogs.oracle.com/shay and search for Glassfish
